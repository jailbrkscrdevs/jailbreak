## 🔧 Troubleshooting & Common Issues

### ❌ Script Won't Inject

**Symptoms:** Executor shows "Failed to attach" or game crashes on inject.

**Solutions:**
1. Run executor as Administrator (Windows) or grant full disk access (macOS)
2. Temporarily disable antivirus — AV software frequently flags executor processes as false positives
3. Ensure the game is fully loaded before attaching (wait for the lobby screen)
4. Try reinstalling your executor with a fresh download

***

### ❌ GUI Doesn't Appear After Execute

**Symptoms:** Script executes without error but no menu appears in-game.

**Solutions:**
1. Press `Right Shift` to toggle menu visibility manually
2. Check that your screen resolution is at least 1280×720 — lower resolutions may push the GUI offscreen
3. Verify you're using the correct script file (not an outdated cached version)
4. Try executing the script a second time if the first execution was during a loading screen

***

### ❌ Auto Farm Stops Randomly

**Symptoms:** Farm session halts after a few minutes without user input.

**Solutions:**
1. Increase `autofarm.interval_ms` to `2000` to reduce server-side detection
2. Enable `pause_on_player_nearby` in config to let the script pause when others approach
3. Ensure your internet connection is stable — disconnections silently kill farm loops
4. Check `safety.session_limit_minutes` — your session cap may have been reached

***

### ❌ ESP Not Rendering

**Symptoms:** ESP toggle is on but no overlays appear on other players.

**Solutions:**
1. Reduce `esp.distance_limit` — very high values may cause rendering timeouts
2. Toggle ESP off and back on from the in-game menu
3. Some servers have anti-ESP measures; switch to a different server
4. Confirm your executor version supports drawing APIs
